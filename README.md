## Technical Pipeline
<img width="3159" height="1572" alt="technical-pipeline" src="https://github.com/user-attachments/assets/3049c554-6869-4272-8578-5b315e10785f" />

- SLP configures the story with prepared templates, target sounds, and customized target words.
- For home practice, the story is served with multimodal features.
- Child records and interacts with the system and practices words outside of the clinic.
- Analyzed information such as transcription, phonemic score labels, is served to the SLP.

### Requirements

- Python 3.8+
- Node.js 16+
- Firebase Cloud Firestore
- OpenAI API Key
